Common Audio and Video Formats
Type Format Description
Audio MP3
  • A popular format with a manageable file size
  • It compresses audio data efficiently, while only losing sound data that's beyond the range of human hearing
WAV
  • An uncompressed format that's a staple in the music industry
  • It's a common format for CDs and offers the purest form of audio
  • WAV files are large in size compared to lossless compressed formats
AIFF
  • An uncompressed format that's compatible with the Mac ecosystem
  • t offers high audio quality and shares many similarities with the WAV format
Video AVI
  • The standard video format for Windows
  • Contains both audio and video
  • Uses a variety of codecs, like DivX and XviD
MP4
  • The standard format for web videos and TV
  • High-quality with relatively small file sizes
WMV
  • A popular and widely used format
  • Offers several advantages, particularly for Windows users
